# ⚡ FF BIO TOOL (THORYXFF)

A complete web tool to create and apply colored bios for Free Fire profile. Users can also **HOST their OWN COPY** of this tool by watching the video tutorial.

## 🔥 Features

| Feature | Description |
|---------|-------------|
| ✨ Colored Bio | Support hex color codes like `[c][FF0000]Red Text` |
| 🎨 Formatting | Bold `[b]`, Italic `[i]`, Underline `[u]` |
| ⚡ V-Badge | Insert colored V badges with one click |
| 🌈 Gradients | Pre-made gradient presets |
| 📱 Mobile Friendly | Responsive design works on all devices |
| 🌍 Region Support | Auto-detects IND, ID, BR, etc. |
| 📋 Templates | Pre-made templates with shayari |
| 🔐 Token Auth | Uses EAT/Access Token for FF API |
| 🎬 DIY Video | Tutorial to create your own website |

---

## 🚀 FOR USERS - How to Use

### Step 1: Get Your EAT Token
- Go to: https://ticket.kiosgamer.co.id/
- Login with your Free Fire account
- Copy your EAT/Access Token

### Step 2: Use the Tool
- Paste your token in the input box
- Your UID and Region will auto-load
- Type your bio or pick a template
- Click **"Apply Bio to Free Fire"**

### Step 3: Check Your Profile
- Open Free Fire
- Go to your profile
- Bio updated instantly!
